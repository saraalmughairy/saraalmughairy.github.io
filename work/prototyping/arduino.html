Five Arduino Builds

In effort to learn the basics of EE, I built a light sensor, binary LED counter, magnet sensor, doorbell, and a dimmer switch.

Overall, I found this assignment to be a great first step to physical computing. While I had previously found it daunting, I feel like I now have some of the fundamentals down, and at least feel comfortable ‘messing around’ and looking things up until I figure it out. I enjoyed being able to test the Hall Effect and light sensors in real time, and think my next project I decided to pursue with arduino will have an interactive component as well. Some of the sketches I found online seemed to utilize LED lights in place of a serial log by having the bulbs light up if a magnet (or light) was detected. This seemed like it made less sense for a light sensor, which is an analog input, because the output is not binary and so you lose the nuanced reading that you otherwise get with a serial log.

Light Sensor

Implement a light sensor that could assess and report the level of brightness in the room.

I found instructions for the build and associated sketch here. I then removed the code from the sketch for implementing LED lights. I tested the effectiveness of the device by using my hand to cover varying amounts of the sensor.

Binary LED Counter

Use three LED lights to create a counter, using blinking patterns to indicate which number the sequence was on.

I found instructions for the build and associated sketch here. I then added a print statement for each light pattern so that I could see in the serial log which number was being reflected by the light pattern.

Magnet Sensor

Implement a magnet sensor that could sense and report when a magnet was held up to it.

I found instructions for the build and associated sketch here. I then removed the code from the sketch for implementing LED lights, and added a print statement so that I could see in the serial log when the sensor detected a magnet. I tested the sensor by seeing whether the serial output changed when a magnet was held up to it, and it did, from reading zero to reading one.
