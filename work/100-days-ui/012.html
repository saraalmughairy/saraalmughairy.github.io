012. Product Page

Design an e-commerce shop for a single item.

For this product page UI, I wanted to create an experience that was proportional for a smartphone, and realistically usable at that scale. I decided to feature the product image front-and-center, and to have text and review details available upon scrolling. The user can see the product first, and learn more about it if they are interested. The product image scales down to make enough room for easily readable text.


Tools used: Sketch, Principle, Adobe PS
