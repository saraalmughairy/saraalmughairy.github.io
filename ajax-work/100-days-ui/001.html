001. Sign Up

Design a sign up page, modal, form, app screen, etc. (It's up to you!)

We never got around to making a login page for Ramble, so this seemed like an obvious choice. The sensitive nature of Ramble makes it so that users might want to be logged out after every use. For this reason, I thought touch-id would be a desirable feature for quick login. This was my first time ever using Principle to animate my designs, so in retrospect, I'm realizing I got a little too trigger happy with the motion graphics :)

Tools used: Sketch, Principle, Adobe Photoshop
